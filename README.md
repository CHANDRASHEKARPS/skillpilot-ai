# SkillPilot AI

SkillPilot AI is an adaptive learning platform that generates personalized learning roadmaps using AI.

The system helps users learn skills through structured study plans, interactive quizzes, adaptive guidance, and AI-based learning assistance.

---

## Features

* Personalized AI-generated learning roadmaps
* Interactive MCQ quiz system
* Adaptive AI study coach
* Progress tracking
* Dynamic roadmap adjustment
* Learning memory system
* Resource and video recommendations
* Excel roadmap export
* Roadmap poster generation

---

## Technology Stack

Frontend:

* Streamlit

Backend:

* Python

AI:

* Groq LLM API

Libraries:

* OpenPyXL
* Pillow
